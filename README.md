# Vivaan Web Media

Professional website for Vivaan Web Media, a Springfield, Illinois based creative and digital services business offering web design, SEO, digital marketing, AI business solutions, commercial photography, branding, and website maintenance.

## About This Website

This site is built as a fast static website using HTML, CSS, and JavaScript. It is ready for GitHub Pages hosting and includes service sections, portfolio galleries, contact information, service areas, and a business growth blog.

## Main Features

- Responsive homepage for desktop, tablet, and mobile
- Web design, SEO, digital marketing, AI, and commercial photography service sections
- Portfolio gallery with real commercial and product photography examples
- Blog section with 64 article cards
- Modal article reader for blog content
- Contact form and business contact details
- GitHub Pages ready file structure

## GitHub Pages Setup

1. Upload all files to the main/root area of the GitHub repository.
2. Open the repository settings.
3. Go to Pages.
4. Set the source to the main branch and root folder.
5. Save and wait for GitHub Pages to publish the site.
